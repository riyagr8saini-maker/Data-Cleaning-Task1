# Task 1 - Data Cleaning with Pandas

## Overview
This task demonstrates data exploration and cleaning using Pandas on the shopping dataset.

## Steps Performed
- Loaded dataset into Pandas.
- Explored shape, columns, and data types.
- Handled missing values by filling with mean or dropping rows.
- Filtered rows and selected specific columns.
- Removed duplicate entries.
- Created a derived column `total_amount`.
- Saved the cleaned dataset as `shopping_cleaned.csv`.

## Files
- `Combined_dataset.csv` → Original dataset
- `data_cleaning.ipynb` → Jupyter Notebook with code and explanations
- `shopping_cleaned.csv` → Cleaned dataset
- `README.md` → Summary of work
